# Management-and-monitoring-system-for-different-electronic-devices
This is a project developed in Java that simulates a management and monitoring system for different electronic devices (such as tablets, smartphones, laptops, etc.). The program operates through an interactive console interface and allows keeping a detailed record of each device's usage.

The system features an interactive menu that allows the user to perform the following actions:

1. **Register a device:** Allows registering a new device by entering its unique code, username, and device type.
2. **Register a usage session:** Adds an activity record to a specific device, saving data such as hours of use, energy consumption, and the maximum temperature reached during the session.
3. **Show device information:** Searches for a device by its code and displays its detailed information, including its session history and individual metrics.
4. **List all devices:** Prints on screen all the devices registered in the system.
5. **Show global statistics:** Calculates and displays general system data, such as the global average energy consumption and which is the most used device (with the most hours of use).
6. **Exit:** Closes the program safely.
